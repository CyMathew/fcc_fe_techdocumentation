Introduction

Go is a new language made by Google. It borrows ideas from existing languages and is most similar to C++

Go specializes in concurrency.

Declaring Variables and Functions

Variables in Go are declared in a manner similar to C++ but it aims to simplify the declaration process by allowing the user to type in a stream of thought.

                    x int
                    p *int
                    a [3]int

Here a variable x name is declared first before specifying its type. Same with the a pointer p, and an array a. Note that the number of elements is described before the type

Functions are declared like so:

                    func main(variable int) int {
                        return 0
                    }

The keyword func is used to declare a function with its return type also being stated at the end of the declaration.

Comments & Style Warnings

Go's comment style is same as C++ which means multi-line comments are opened with /* and closed with */

Single line comments are opened with //

                    /*
                     * Multiline comment
                     */

                    // Single Line Comment

As you may have noticed by now, Go does not require semicolons in the source code. It is auto added by the compiler but a consequence of that "feature" is that brackets must begin right after a statement that requires it:

                    if i < f() {
                        g()
                    }
                
                
                    
                    This is invalid:
                    if i < f()  // wrong!
                    {           // wrong!
                        g()
                    }

Control Statements

A small sample of all the control statements provided by Go:

  • For
  • If

Structs

Structs are types that contain named fields. Used to create "objects" of sorts. Go doesn't have actual objects in the language

                    type T struct {
                        variableName1 string
                        variableName2 int
                    }
